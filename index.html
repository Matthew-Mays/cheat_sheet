Website Building Cheat Sheet

HTML References

Table Tags
Tag Name Description
<table> Table The wrapper element for all HTML tables.
<caption> Caption Sets a caption title above the table.
<thead> Table Head Designates a group of rows as the header.
<tbody> Table Body Designates a group of rows as the body.
<tfoot> Table Footer Designates a group of rows as the footer.
<tr> Table Row The table row container.
<th> Header Cell Contains HTML table header information.
<td> Table Data Contains HTML table data.
